# Santanu Sarker | Professional Portfolio

A premium, responsive portfolio website featuring glassmorphism, fluidic animations, and interactive 3D effects.

## Features
- **Fluidic Interactions**: Smooth scroll spying and backdrop blur effects.
- **3D Hero Tilt**: Interactive mouse-tracking tilt effect on the hero image.
- **Dynamic Content**: Data-driven components for projects and professional timeline.
- **Glassmorphic UI**: High-end aesthetic using modern CSS backdrop filters.

## Local Development
To run this project locally, you can use any static file server.

If you have Node.js installed:
```bash
npm start
```

Or using Python:
```bash
python3 -m http.server 3000
```
